Getting Started

Welcome to the PennyWise financial manager! This website will allow you to manage your finances and put the IT in your budget!

To start off, create an account using the "Create a new account" button at the login screen here. After creating a new account, you can log in at any time via the login screen with the credentials that you made.

Navigating

PennyWise has six tabs to go between:

  • Overview : Provides an overview of all your finances
  • Transactions : Allows you to log your latest spending and income
  • Bills : Allows you to log your upcoming bills
  • Budget : The budget tab allows you to monitor your spending for certain categories
  • Investments : In the investments tab, you can track investments such as stocks
  • Property : The property tab allows you to keep track of your property such as real estate and vehicles
